# Contributing

This repository holds an **actively maintained research publication** — an
IEEE-format paper, an M.Sc. research report, and the accompanying presentation.
Factual corrections that keep the record accurate are welcome.

## Reporting

- **Security or integrity issues:** follow [`SECURITY.md`](SECURITY.md).
- **Factual corrections, errata, and broken links:** open a concise issue via
  GitHub Issues (enabled), or contact the author below.

Please raise substantive changes as an issue first, so the correction can be
discussed and folded into the published record through the normal
camera-ready / erratum process.

## Citation

If you reference this work, please cite it via the metadata in
[`CITATION.cff`](CITATION.cff).
