\chapter{Schematics and Layouts}
\label{app:schematics}

This appendix contains the detailed schematic diagrams and Printed Circuit Board (PCB) layouts for the retrofitted test-diagnosis system modules. It also includes photographic evidence of the inherited Rev 0.0 boards and the corrected, unpopulated Rev 1.0 boards.

\section{Connectors}
\label{sec:app_connectors}
Key connector systems used include standard pin headers for board-to-board connections and specific parts like TE's AMPMODU System 50 family for compact connections, ensuring robust interconnectivity between the modular boards~\cite{te_ampmodu_sys50}.
