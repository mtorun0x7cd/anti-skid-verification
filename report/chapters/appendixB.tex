\section{Design Rule Check (DRC) Summary}
\label{sec:app_drc}

Design Rule Checks (DRC) were performed using the Autodesk Eagle EDA software for all revised PCB layouts. The rules enforced are a conservative superset of the constraints provided by typical PCB manufacturers, ensuring high yield and reliability. All Rev 1.0 layouts passed the defined DRC process with zero errors. The key electrical and clearance constraints that were verified are listed below:

\begin{itemize}
    \item \textbf{Layer Setup:} 4-layer stackup (Top, Ground, Power, Bottom).
    \item \textbf{Minimum Track Width:} \texttt{8mil} (\texttt{0.2032mm}) for general signal routing.
    \item \textbf{Minimum Track Spacing (Clearance):} \texttt{8mil} (\texttt{0.2032mm}) between copper features (track-to-track, track-to-pad).
    \item \textbf{Minimum Drill Size:} \texttt{16mil} (\texttt{0.4mm}) for standard vias and through-holes.
    \item \textbf{Minimum Annular Ring Width:} \texttt{6mil} (\texttt{0.1524mm}) to ensure a solid connection between a drill hole and its surrounding copper pad.
    \item \textbf{Via Specification:}
        \begin{itemize}
            \item Via Drill Size: \texttt{16mil} (\texttt{0.4mm})
            \item Via Pad Diameter: \texttt{28mil} (\texttt{0.7112mm})
        \end{itemize}
    \item \textbf{Polygon Isolation:} \texttt{12mil} (\texttt{0.3048mm}) for clearance between ground/power planes and signal traces.
    \item \textbf{Solder Mask "Swell":} \texttt{2mil} (\texttt{0.0508mm}) expansion to ensure pads are fully clear of solder mask.
\end{itemize}

The successful completion of the DRC verifies that the PCB layouts are physically manufacturable and conform to established best practices for signal integrity and electrical reliability.
